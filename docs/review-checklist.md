# Review Checklist

## 领域规则

- 一个被邀请人只能绑定一个邀请人。
- 首次有效消费只计一次，重复订单核销按幂等返回。
- 新客礼只在首次有效消费发放。
- 邀请达标券按已到账规则去重，不重复发放。

## 权限与安全

- 店员只能核销和查询，不能改奖励规则。
- 老板才能改规则、调关系、管员工、看审计日志。
- 敏感写操作全部经过云函数，不允许客户端直写数据库。
- 手机号唯一，员工账号唯一，邀请关系改动必须落审计日志。

## 上线前

- 小程序首页、规则页、隐私指引文案一致。
- 体验版账号和正式版账号分离。
- 奖励规则、有效期、店内可兑菜品和核销流程已经过门店演练。
- 审核截图、测试路径、客服联系方式已准备完成。
